Important FAQ

What is the Document Object Model (DOM)?

ANSWER: The Document Object Model (DOM) is a programming interface used for web documents that represent the structure of an HTML page as a tree of objects.

How do you select one element or more elements by DOM?

ANSWER:
For single element select, using two types:
1. document.getElementById("");
2. document.querySelector("");

For many elements select, using three types:
1. document.getElementsByTagName("");
2. document.getElementsByClassNam("");
3. document.querySelectorAll("");

What is appendChild() in JavaScript?

ANSWER: The appendChild() method in JavaScript used to add a new node to an exiting parent node. It is generally used to dynamically modify the structure of an HTML document.

What is showModal() method?

ANSWER: The showModal() method is part of the HTMLDialogElement API and is used to display a dialog box in a modal state.
